# ✨️ Star-Clock
🚀️ It's simply a clock app which provide the user current time and greeting as per your current location and this application also provide motivational quotes by using Quote API.


## 🚀️ Demo :
[Star Clock](https://atultrp.github.io/Star-Clock/)
![Screenshot from 2021-10-13 10-41-01](https://user-images.githubusercontent.com/69186169/137071165-064fa444-d7f7-48b8-888b-0fd59027f2ad.png)
